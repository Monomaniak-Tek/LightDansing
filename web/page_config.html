Configuration en lot du groupe

Allumer uniquement les recepteurs a configurer, puis choisir le groupe cible :

Aide: configurer 1 récepteur individuellement (bouton récepteur)

  1. Allumer le récepteur normalement.
  2. Maintenir le bouton récepteur environ 2 secondes.
  3. Le bandeau défile les couleurs de groupe (1 couleur par seconde).
  4. Quand la bonne couleur apparaît, appuyer une 2e fois pour valider.
  5. Le bandeau clignote: le groupe est sauvegardé (même après reboot).
